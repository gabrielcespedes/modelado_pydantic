# Modelado con Pydantic

Ejemplo enfocado en el modelado y validación de datos
utilizando Pydantic en Python.

Se implementan ejemplos simples para trabajar con modelos tipados,
validación automática de datos y estructuras limpias.

------------------------------------------------------------------------

## Descripción

El proyecto contiene scripts básicos que permiten:

-   Definir modelos de datos utilizando Pydantic
-   Validar información de entrada
-   Trabajar con estructuras de datos tipadas
-   Ejecutar ejemplos prácticos desde un archivo principal

------------------------------------------------------------------------

## Tecnologías

-   Python 3.x
-   Pydantic
